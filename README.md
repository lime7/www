# Моя Front-End сборка


###СОЗДАНИЕ ИСХОДНИКОВ:
```
mkdir www
cd !$
mkdir app/{css,js} –p
cd app
touch index.html js/{main.js,plugins.js} css/{base.css,style.css} 
cd ..
```






### Установка:
```
$ git clone https://github.com/lime7/www.git
```

```
$ npm i
```

```
$ bower i
```

```
$ gulp
```
